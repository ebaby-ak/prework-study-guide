HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • External Style Sheet is the most common way to style a webpage with CSS.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Selector: applies rules to all elements visible on the page.
  • Separation of Concerns: separate code into different files based on use.

Git

  • git status: checks what main branch we are currently on
  • git checkout -b feature/add-name: creates a new branch and switches to it
  • git add -A: stage all files and add changes
  • git commit -m "message": describes what commit contains
  • git checkout main: switch to main branch
  • git pull origin main: check local branch sync with base branch
  • git push origin feature/add-name: push changes to remote branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Variable Keywords:
  • - const: create variables that can't be reassigned a value.
  • - let: assign specific data that we want stored in our variable.
  • - console.log: method to output a message to web console by adding an argument.
  • To link script.js to index.html: under footer before body closing.
  • Control flow is the order in which a computer executes code in a script.
  • ===: Strict Equality Operator, checkss to see if two values are equal. Boolean result "True" if equal, "Flase" if not.
  • Array: single variable used to hold a group of data.
  • JS & other programming languages are zero-indexed + sequential.
  • loops:
  • - for: uses predictable pattern of indices to perform a task on all items in an array by allowing single code block to be executed repeatedly. Needs index & array length.
  • 1st Statement determines starting point for loop. ex. let x=0
  • 2nd Statement is the condition. ex. x less than shapes.length
  • 3rd Statement allows array to iterate over each item. ex. x++
  • Commenting Out: makes the conditional statement inactive without deleting it. In JS use /* before and */ after code block.
  • Function: set of instructions that tells computer how to perform a certain task. Must call function by name to execute it.
